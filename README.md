# 🌟 Pixel  🌟

<p align="center">
  <strong>✨ Capture your moments with us! ✨</strong><br>
  A retro, Y2K-inspired digital arcade photobooth built with a flashy, game-like aesthetic. 
</p>

---

## Features

* **⚡ Retro Star Matrix UI:** Dynamic, moving matrix background that brings arcade nostalgia straight to your browser.
* **📸 Custom Photostrip Generation:** Snap your pictures and compile them seamlessly into a classic digital filmstrip layout.
* **💖 Girly Y2K Aesthetic:** Complete with pixel typography, vibrant pink/neon themes, custom retro mouse cursors, and 3D arcade-style UI elements.
* **💾 Instant Downloads:** Save your custom compiled photostrips straight to your local device with a single click.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Canvas Manipulation:** HTML5 Canvas API for photostrip image layering, scaling, and generation
* **Media Operations:** Integrated media flows for handling image assets and background rendering
